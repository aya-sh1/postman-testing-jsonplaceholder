# postman-testing-jsonplaceholder

## 📌 Project Overview
This project demonstrates automated API testing for JSONPlaceholder using Postman. 

## 🔍 Test Coverage
- **Functional:** Schema validation, data integrity, and status codes.
- **Performance:** Response time monitoring (Threshold: < 400ms).
- **Load Testing:** Evaluating system behavior under sustained concurrent traffic for 10 minutes.

## 🛠️ Tools
- **Postman Ecosystem:** - Collections & Environments (Test organization)
   - **Performance Tool** (Load testing & metrics)
- **JavaScript** (Chai Assertion Library)
